## About

This repository is the master index of my marketing analytics work. Each project solves a real business problem: diagnosing poor lead quality, segmenting customers by value, evaluating experiments, or auditing paid channel spend. All projects use real analytical methods on realistic synthetic data and include a business recommendation layer — not just code.

## How to Navigate This Portfolio

Each repository follows the same structure:

```
/data          → synthetic or anonymised sample data
/notebooks     → Python notebooks or SQL scripts
/outputs       → charts, dashboards, slide decks
README.md      → Problem → Approach → Findings → Recommendation
```

**Start here if you are:**
- A **hiring manager or recruiter** → Read the README of each project first. Each one opens with the business context in plain English.
- A **marketing lead** → Look at the outputs/ folder and slide decks for each project.
- A **technical reviewer** → Open the notebooks/ folder for code, methodology, and data.
